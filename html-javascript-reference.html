HTML Elements

1. H1 (Heading)

Correct Syntax:

<h1 id="main-title" class="header" title="Page Title">Welcome to My Website</h1>

Most Popular Attributes:

Attribute Description
id Unique identifier for the element
class CSS class name(s) for styling
style Inline CSS styles
title Additional information (tooltip)
contenteditable Makes content editable (true/false)

2. P (Paragraph)

Correct Syntax:

<p id="intro" class="text-content" style="color: blue;">This is a paragraph.</p>

Most Popular Attributes:

Attribute Description
id Unique identifier
class CSS class name(s)
style Inline CSS styles
contenteditable Makes content editable

3. Div (Container)

Correct Syntax:

<div id="container" class="wrapper" data-user="123"> <p>Content goes here</p> </div>

Most Popular Attributes:

Attribute Description
id Unique identifier
class CSS class name(s)
data-* Custom data attributes (data-name="value")
role ARIA role for accessibility
draggable Enables drag functionality

4. Table

Correct Syntax:

<table id="data-table" class="styled-table"> <thead> <tr><th scope="col">Name</th><th>Age</th></tr> </thead> <tbody> <tr><td>John</td><td>25</td></tr> </tbody> </table>

Related Elements & Attributes:

  • <thead> - Table header section
  • <tbody> - Table body section
  • <tr> - Table row
  • <th> - Header cell (scope, colspan, rowspan)
  • <td> - Data cell (colspan, rowspan)

5. A (Anchor/Link)

Correct Syntax:

<a href="https://example.com" target="_blank" rel="noopener">Visit Site</a>

Most Popular Attributes:

Attribute Description
href REQUIRED URL destination
target _blank (new tab), _self (same window)
rel noopener (security), noreferrer, nofollow
download Download file instead of navigate
title Link description/tooltip

6. Img (Image)

Correct Syntax:

<img src="logo.png" alt="Company Logo" width="200" height="100" loading="lazy">

Most Popular Attributes:

Attribute Description
src REQUIRED Image source URL
alt REQUIRED Alternative text for accessibility
width / height Image dimensions in pixels
loading lazy (load when visible), eager (load immediately)
srcset Responsive image sources

7. Form

Correct Syntax:

<form action="/submit" method="POST" enctype="multipart/form-data"> <input type="text" name="username"> <button type="submit">Submit</button> </form>

Most Popular Attributes:

Attribute Description
action URL to submit form data
method GET (URL params) or POST (request body)
enctype multipart/form-data (for file uploads)
autocomplete Enable autocomplete (on, off)
novalidate Disable validation

8. Input

Correct Syntax Examples:

<input type="text" name="username" placeholder="Enter name" required> <input type="email" name="email" pattern="[a-z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,}$"> <input type="checkbox" name="agree" checked> <input type="file" name="upload" accept=".pdf,.doc">

Most Popular Attributes:

Attribute Description
type text, password, email, number, checkbox, radio, file, date, etc.
name Input name for form submission
value Input value
placeholder Hint text shown when empty
required Makes field mandatory (boolean)
disabled Disables the input
maxlength Maximum character length
pattern Regular expression for validation

9. Select (Dropdown)

Correct Syntax:

<select name="country" required> <option value="">Choose...</option> <option value="us" selected>United States</option> <option value="uk">United Kingdom</option> </select>

Most Popular Attributes:

  • name - Name for form submission
  • required - Makes selection mandatory
  • multiple - Allow multiple selections
  • Related: <option> (value, selected, disabled)

10. Button

Correct Syntax:

<button type="submit" name="action" value="save">Save</button>

Most Popular Attributes:

  • type - button (default), submit, reset
  • disabled - Disables the button
  • onclick - JavaScript click handler
  • name / value - For form submission

JavaScript Concepts

11. Arrays

Correct Syntax:

const arr = [1, 2, 3, 4, 5]; arr.push(6); // [1,2,3,4,5,6] arr.pop(); // [1,2,3,4,5] arr.map(x => x * 2); // [2,4,6,8,10] arr.filter(x => x > 3); // [4,5]

Most Popular Methods:

  • push(), pop() - Add/remove from end
  • shift(), unshift() - Add/remove from beginning
  • map() - Transform each element
  • filter() - Select elements by condition
  • reduce() - Reduce to single value
  • forEach() - Execute function for each element
  • find(), includes() - Search elements
  • sort(), reverse() - Reorder elements
  • concat(), join() - Combine/convert arrays

12. Loops

Correct Syntax:

// For loop for (let i = 0; i < 5; i++) { console.log(i); } // For...of loop for (let item of array) { console.log(item); } // While loop while (condition) { // code }

Loop Types:

  • for - Traditional counter loop
  • for...of - Iterate array values
  • for...in - Iterate object keys
  • while - Loop while condition is true
  • do...while - Execute at least once
  • break, continue - Loop control statements

13. Conditions

Correct Syntax:

if (age >= 18) { console.log("Adult"); } else if (age >= 13) { console.log("Teen"); } else { console.log("Child"); } // Ternary operator const result = age >= 18 ? "Adult" : "Minor";

Operators:

  • ===, !== - Strict equality/inequality
  • >, <, >=, <= - Comparison operators
  • &&, ||, ! - Logical AND, OR, NOT
  • ?? - Nullish coalescing operator

14. Functions

Correct Syntax:

// Function declaration function greet(name) { return "Hello " + name; } // Arrow function const greet = (name) => "Hello " + name; // Function with default parameters function greet(name = "Guest") { return "Hello " + name; }

Key Concepts:

  • Function declaration - function name() {}
  • Arrow function - const name = () => {}
  • Parameters & return - Input and output values
  • Default parameters - Fallback values
  • Callback functions - Functions as arguments
  • Closure - Access to outer scope

15. Class

Correct Syntax:

class Person { constructor(name, age) { this.name = name; this.age = age; } greet() { return `Hi, I'm ${this.name}`; } } const person = new Person("John", 25);

Key Components:

  • constructor - Initialization method
  • this - Refers to current instance
  • Methods - Instance methods
  • Static methods - Called on class itself
  • extends - Inheritance from parent class
  • super - Call parent constructor/methods

16. Objects

Correct Syntax:

const person = { name: "John", age: 25, greet() { return "Hello"; } }; person.name // Dot notation person["name"] // Bracket notation Object.keys(person) // ["name", "age", "greet"]

Most Popular Methods:

  • Object.keys() - Get property names
  • Object.values() - Get property values
  • Object.entries() - Get [key, value] pairs
  • Object.assign() - Copy properties
  • Object.freeze() - Prevent modifications
  • Object.hasOwn() - Check property existence
